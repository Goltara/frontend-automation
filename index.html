What is Grunt?

  • JavaScript task runner
  • Runs on Node.js
  • Helps automate tasks
    • Code linting, minification, concatenation
    • Image optimization & minification
    • CSS pre/post-processing
    • CSS Sprites
    • Etc, etc.
